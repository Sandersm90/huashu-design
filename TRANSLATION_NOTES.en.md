# English Documentation Notes

This repo now includes parallel English documentation files alongside the original Chinese files.

## Naming
- `SKILL.en.md` mirrors `SKILL.md`
- `references/*.en.md` mirrors `references/*.md`

## Translation strategy
- Keep structure, section order, tables, and code examples as close to the source as practical.
- Prefer natural English, but preserve repo-specific terms such as “Huashu-Design”, “Tweaks”, “Junior Designer mode”, and “AI slop”.
- Preserve commands, code, filenames, and technical constraints exactly unless a literal translation would be misleading.
- When a phrase is highly idiomatic in Chinese, translate the intent rather than the literal wording.

## Scope
This pass is intended as a working English companion for non-Chinese readers, not a fresh rewrite of the docs.
